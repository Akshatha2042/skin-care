#  Skin Care Guide

A simple skincare web application built using HTML, CSS, and JavaScript.

## Features

* Oily Skin Routine
* Dry Skin Routine
* Combination Skin Routine
* Normal Skin Routine
* AM and PM skincare recommendations

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download the project files.
2. Open `index.html` in your browser.
3. Select your skin type to view the skincare routine.
